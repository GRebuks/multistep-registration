# multistep-registration

## Dependencies  
+ Node package manager (npm)
+ PHP 8.1.0 (or greater)
+ Composer 2.2.0 (or greater)

```bash
php artisan update
php artisan install
```

```bash
npm install
```

```bash
php artisan key:generate
php artisan migrate
```

```bash
php artisan serve
npm run dev
```
