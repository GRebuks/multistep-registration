# multistep-registration

## Instalācija  
+ Node package manager (npm)
+ PHP 8.1.0 (vai lielāks)
+ Composer 2.2.0 (vai lielāks)


```bash
php artisan update
php artisan install
```

```bash
npm install
```

```bash
php artisan key:generate
php artisan migrate
```

```bash
npm run dev
php artisan serve
```

Pēc komandu izpildes lapa pieejama localhost:8000
